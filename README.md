      Projeto sem nome

- Um voluntário deve poder se cadastrar no site
- Uma ong deve poder se cadastrar no site
- Um voluntário deve poder pesquisar por trabalhos perto dele
- Um voluntário deve poder pesquisar por trabalhos pela localidade
- Um voluntário deve poder pesquisar por trabalhos pela sua causa (ambiental, humanitária, animais)
- Um voluntário deve poder pesquisar por trabalhos na sua área de atuação
- Um voluntário deve poder pesquisar por funções específicas
- Um voluntário deve poder filtrar pesquisa pelo tempo que ele tem disponível
- Um voluntário deve poder filtrar pesquisa pelo horário que ele tem disponível
- Uma ong deve poder divulgar um trabalho precisando de voluntário
- Um voluntário deve ver o perfil de uma ong
- Um voluntário deve poder pesquisar trabalho de uma determinada ong
- O voluntariado deve poder pesquisar por um trabalho de um dia
